Meta-meta: Descrição Semântica

A descrição semântica aqui apresentada será passada através de exemplos. Comentários, quando necessários, acompanharão o código e seguirão a definição de <comment-line>.

<variable-declaration>

A declaração de variável, ainda que não faça inicialização, considera múltiplas variáveis, estas quais podem ser definidas também como arrays. Um exemplo de declaração de variável, considerando o identificador "number" como um tipo atômico, seria como segue:

number a;
number b: 0; // Declara b e inicializa com 0
number resultado,vetor[128];

<value-assignment>

Atribuição de valores à uma variável em meta-meta pode ser dada através da seguinte forma:

number a: 2; // atribui 16 à variável "a"
number b: 32; // atribui 32 à variável "b"
number resultado,vetor[128]: 64; // inicializa o vetor inteiro com o valor 64
resultado: a*b+vetor[0]; // atribui a*b+vetor[0] <=> 2*32+64 = 128 à variável resultado

<function-declaration> & <explicit-return>

A declaração de função, considerando a existência do tipo atômico "number" é feita da seguinte forma:

number getTime();
number sqrt(number x);

O retorno de valor em uma função é realizado utilizando um comando apenas com um valor ou explicitamente com o comando "return":

number getPI(): 3.141592653589793.
number getPI(): return 3.141592653589793.

<function-definition>

A definição da função pode ser feita omitindo os tipos de retorno e des seus parâmetros. Por exemplo:

number f(number x);
f(x): x*x.
